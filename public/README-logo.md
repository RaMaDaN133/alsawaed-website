# شعار الموقع (Logo)

ضع ملف شعار المكتب هنا باسم **logo.png** بالضبط:

```
D:\website\public\logo.png
```

بمجرد حفظ الملف بهذا الاسم في هذا المجلد:

- سيظهر الشعار تلقائيًا في **الهيدر** (أعلى كل الصفحات).
- وسيظهر أيضًا في **تبويب المتصفح (favicon)**.

> قبل إضافة الملف، يعرض الهيدر أيقونة ذهبية مؤقتة بدل اللوجو حتى لا تظهر صورة مكسورة.

نصائح:
- يُفضّل صورة مربعة أو قريبة من المربعة، وبخلفية شفافة (PNG transparent).
- مقاس مناسب: 256×256 بكسل أو أكبر.
- لو أردت صورة favicon مربعة منفصلة بجودة أعلى، يمكن إضافتها لاحقًا.
