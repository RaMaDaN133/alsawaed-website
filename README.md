# مكتب السواعد للمحاماة والاستشارات القانونية والتحكيم

موقع ويب احترافي متكامل لمكتب محاماة، مبني باللغة العربية (RTL) بتصميم حديث وفخم.

## 🛠️ التقنيات المستخدمة

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (للحركات والانتقالات)
- **React Icons**
- خطوط عربية احترافية: **Cairo** و **Tajawal** (عبر `next/font`)

## 🚀 التشغيل

```bash
# 1. تثبيت الحزم
npm install

# 2. تشغيل بيئة التطوير
npm run dev
# الموقع على: http://localhost:3000

# 3. بناء نسخة الإنتاج
npm run build
npm start
```

## ✏️ التخصيص

### تعديل بيانات المكتب
كل المحتوى في ملف واحد: **`lib/data.ts`**
- `siteConfig` — الاسم، الهاتف، البريد، العنوان، وسائل التواصل
- `services` — الخدمات القانونية
- `team` — أعضاء الفريق
- `testimonials` — آراء العملاء
- `faqs` — الأسئلة الشائعة
- `articles` — المقالات

### استبدال صور الفريق
الصور حاليًا عبارة عن **Placeholder** قابلة للاستبدال بسهولة دون أي تعديل على التصميم.
في `lib/data.ts`، غيّر قيمة `image` لكل عضو إلى مسار صورتك:

```ts
{
  id: 1,
  name: "أ. عبدالله السواعد",
  image: "/team/abdullah.jpg", // ضع الصورة في مجلد public/team
  ...
}
```

### خريطة Google Maps
في `components/sections/MapPlaceholder.tsx` استبدل رابط `iframe` بكود التضمين
الخاص بموقعك من خرائط Google.

### نموذج التواصل (Resend)
نموذج التواصل يرسل إيميلًا فعليًا عبر **Resend** (بدون قاعدة بيانات) من خلال
Route Handler في `app/api/contact/route.ts`، مع تحقق من البيانات على الخادم
ورسائل نجاح/خطأ للمستخدم.

**الإعداد:**
1. انسخ `.env.local.example` إلى `.env.local`.
2. املأ المتغيرات:
   - `RESEND_API_KEY` — من [resend.com/api-keys](https://resend.com/api-keys)
   - `CONTACT_EMAIL` — البريد الذي تصل إليه الرسائل
   - `CONTACT_FROM` (اختياري) — عنوان مُرسِل من نطاق موثّق في Resend (وإلا يُستخدم نطاق الاختبار)
3. أعد تشغيل الخادم.

> ملف `.env.local` مُتجاهَل في Git؛ لا تضع المفاتيح في الكود.
> في وضع الاختبار، Resend يسلّم فقط إلى بريد صاحب الحساب — للإنتاج وثّق نطاقك واضبط `CONTACT_FROM`.
